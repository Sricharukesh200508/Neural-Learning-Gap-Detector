Setup & Deployment

1. Backend API Start Sequence

Ensure you have Python 3.10+ installed. The neural engine serves as the core intelligence layer.

cd backend python -m venv myenv .\myenv\Scripts\activate # Windows pip install -r requirements.txt python main.py

The API will be available at http://localhost:8000.

2. Frontend Launch Sequence

Ensure you have Node.js 20+ installed. The immersive dashboard provides deep insights.

cd frontend npm install npm run dev

The Dashboard will be accessible at http://localhost:3000.
